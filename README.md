
# App QR con Verificación por URL — Colegio de Guías de Turistas de Chiapas A.C.
Python 3.13.9 + dependencias compatibles. Sin cámara embebida; el QR contiene la URL con el token.

## Deploy
- Subir al repo y en Streamlit Cloud usar `app.py` como Main file.
- Ajustar `base_url` en la pestaña **Ajustes** o en `st.secrets`.
